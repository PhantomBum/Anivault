AniVault Unvaulted — changelog

1.0.6
- **UI:** **AniVault** text logotype (Ani + Vault styling) replaces the diamond icon app-wide — sidebar, home, gate, and window title bar. Tag **`v1.0.6`** → installer on [GitHub Releases](https://github.com/PhantomBum/Anivault/releases/latest).

1.0.5
- **UI:** Restored the official **AniVault rounded-square logo** (SVG assets) across the shell, home hero, and access gate — same mark as the window title bar. Sidebar brand card, frosted header with logo + page title, and refreshed spacing. Tag **`v1.0.5`** → **AniVaultUnvaultedSetup.exe** + updater files on [GitHub Releases](https://github.com/PhantomBum/Anivault/releases).

1.0.4
- **Release / CI:** GitHub Actions no longer uses glob-based upload on Windows (replaced with **`gh release create` / `gh release upload`** + artifact verification). Squirrel **NuGet ID** remains `AniVaultUnvaulted` (no spaces). Tag **`v1.0.4`** publishes **AniVaultUnvaultedSetup.exe**, nupkg, **RELEASES**, and **latest.yml**.

1.0.3
- **Build / release:** Windows Squirrel installer failed on CI because the NuGet package ID cannot contain spaces. Squirrel `name` is now `AniVaultUnvaulted` (display name unchanged). Tag `v1.0.3` re-triggers **AniVaultUnvaultedSetup.exe** on GitHub Releases.

1.0.2
- **Fix:** Series **Details & episodes** no longer spins forever (React Strict Mode–safe load generation, clear stale details, errors shown before loading). **IPC timeouts** (120s) on `getShowDetails` / episode lists so a hung ani-cli call fails with an error instead of blocking the UI. Tag `v1.0.2` triggers CI to attach **AniVaultUnvaultedSetup.exe** and updater files on GitHub Releases.

1.0.1
- **Fix:** Opening a series from **Find shows** or **Discover** now loads reliably (stable `anime` memo + navigation `state`). Tag `v1.0.1` triggers CI to attach **AniVaultUnvaultedSetup.exe** and updater files on GitHub Releases.

1.0.0 (Unvaulted — stable line)
- Product name is now **AniVault Unvaulted** (out of alpha). Version **1.0.0** ships the accumulated phases 15–30 work: settings search, calendar, lists export, offline download queue (progressive URLs), telemetry opt-in, stream tests, shell polish, and more.
- Windows: installer **AniVaultUnvaultedSetup.exe**, app **anivault-unvaulted.exe**; App User Model ID `com.anivault.unvaulted`. Fresh install recommended for users coming from alpha Squirrel builds.
- GitHub Releases publishes stable assets (publisher `prerelease: false`).
