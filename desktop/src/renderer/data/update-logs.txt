AniVault — changelog

1.0.0-alpha.12
- Windows NSIS: assisted installer + folder picker now forced via package.json `build.nsis` (electron-builder defaults `oneClick` to true when the flag is missing — that was hiding the directory page). Use Next after “install for me / all users” to reach “Choose install location”.

1.0.0-alpha.11
- CI: Windows release workflow now uploads AniVaultSetup.exe + latest.yml via GitHub Release assets (forge publish dry-run was failing; no binaries were attached on alpha.10)

1.0.0-alpha.10
- Windows installer is now NSIS: real setup wizard, AniVault icon on installer/uninstaller, choose install folder before install
- Auto-update uses GitHub Releases `latest.yml` + `AniVaultSetup.exe` (Squirrel/nupkg no longer used on Windows)
- Settings: tab bar layout fixed (list height vs triggers — no clipped/ghosted pills)
- If you installed an older Squirrel build, run this setup once to migrate

1.0.0-alpha.8
- Settings: larger tabs and panels; new Appearance tab (UI density); smooth cursor + shell effects always on (documented)
- Welcome spotlight: glass-style blurred poster background (Nexus-inspired)
- Custom cursor: tip aligned with the real pointer; white branded pointer; scale feedback only
- Title bar: AniVault mark on Windows/macOS; shell blur always enabled in layout
- Minor cleanup (unused config wiring in sidebar layout)

1.0.0-alpha.7
- Stream resolution: retries on transient failures, 90s overall cap, HTTP timeouts on AllAnime fetches
- Clearer load errors on the watch page; recovery hints when another episode may work
- Main process logs uncaught errors / unhandled rejections; logs renderer/gpu child exits
- React error boundary with reload if the UI crashes
- IPC failsafe timeout for stream URL resolution
- Auto-update: correct GitHub repo baked into builds; latest.yml included in published release assets
- Known: some titles/episodes may still fail when upstream sources have no playable stream
- Known: rare freezes or crashes may still occur; report steps if reproducible

1.0.0-alpha.3
- Windows build and GitHub Releases publishing

1.0.0-alpha.2
- Dependency install reliability for automated builds

1.0.0-alpha.1
- Windows installer (AniVaultSetup.exe)
- Watch page mini player (progress, transport)
- Settings: update logs
