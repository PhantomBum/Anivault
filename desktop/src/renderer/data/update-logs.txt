AniVault Unvaulted — changelog

1.2.3
- **Fix:** Packaged app could show a **blank black window** — Vite added `crossorigin` on built `index.html` assets; under Electron’s `file://` loader those scripts/styles failed to run, so React never mounted. Stripped `crossorigin` in the renderer build for Electron. Tag **v1.2.3**.

1.2.2
- **Windows installer:** Branded Squirrel setup — `electron-squirrel-startup` (install/update/uninstall lifecycle), Forge `setupIcon` / `loadingGif` / metadata, generated `installer/installing.gif`. Tag **v1.2.2** → **AniVaultUnvaultedSetup.exe** on [GitHub Releases](https://github.com/PhantomBum/Anivault/releases/latest).

1.2.1
- **Fix:** App could show a **black screen** on startup — `AnivaultConfigProvider` called `getWindowControls()` without importing it, throwing `ReferenceError` after config loaded.

1.2.0
- **Playback / window:** Optional **Keep window above others** — `BrowserWindow.setAlwaysOnTop` via IPC (`windowControls.setAlwaysOnTop`). Synced from config on load and when toggled in **Settings → Playback** (helps with mini-player multitasking).
- **Accessibility:** `DialogContent` **`showCloseButton`** — update dialogs and **command palette** no longer render a hidden duplicate close control; focus trap + footer actions behave predictably.
- **Performance:** `cachedGetShowDetails` / `cachedGetEpisodes` **coalesce concurrent** requests; **Rollup `manualChunks`** for react / react-router / i18next in `vite.renderer.config.ts`. See `desktop/docs/performance-notes.md`.

1.1.0
- **UI / roadmap:** Shared **`AvFutureSurface`** component — diagonal gradients + soft orbs (Studio, Community, Gallery, Clips variants). **Community**, **Gallery**, and **Clips** pages are full preview cards (chips, badges, CTA to **Studio** in Settings) instead of a plain placeholder. **Settings → Studio** uses the same surface for consistency.
- **Copy:** Updated route subtitles and `en_US` / `pt_BR` strings for the coming-soon area.

1.0.8
- **Navigation:** Sidebar sections (Library / Tools / Coming soon); `/community`, `/gallery`, `/clips` show a unified coming-soon page; route headings aligned with i18n.
- **Settings:** Tab panels split under `pages/settings/`; new **Privacy & diagnostics** and **Shortcuts & input** tabs; telemetry moved to Privacy; Find-in-settings and tab URLs updated; **Alt+,** cycles tabs.
- **Shell:** Command palette (**Ctrl+Shift+P**) — jump to routes, every Settings tab, resume watch / scroll to player, copy diagnostics; main scroll **restored per route** (sessionStorage); **Alt+Left** history back.
- **Watch:** Breadcrumb (Discover → series → episode); **copy shareable link** with `id` / `mode` / `ep` query args; opening Watch from that URL hydrates the session.
- **Discover / Search:** Discover tab remembered in session; Find shows restores last query in session and adds a **clear** control on the search field.
- **Home:** Continue watching — context menu **Remove from Continue watching** (clears progress for that series).
- **Lists:** **Bulk select** + remove selected.
- **Docs:** `desktop/docs/performance-notes.md` for profiler/bundle reminders.

QoL: last-tab memory, find-in-settings shortcuts, copy link to tab, offline queue refresh, version copy, and more. Tag **`v1.0.7`** → **AniVaultUnvaultedSetup.exe** on [GitHub Releases](https://github.com/PhantomBum/Anivault/releases/latest).
