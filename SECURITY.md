# Security

If you think you’ve found a **security vulnerability** (not “this stream link broke”), please **don’t** open a public issue with exploit details.

- Prefer **[GitHub private vulnerability reporting](https://github.com/PhantomBum/Anivault/security/advisories)** for this repository, if the feature is enabled.
- Otherwise, use **Security → Advisories** on the repo and keep the report as small and reproducible as you can.

This project doesn’t run a bug bounty program. We’ll read reports in good faith and patch what we can; timelines depend on maintainer availability.

**Out of scope for this page:** copyright complaints about streamed content—that’s between users, hosts, and rightsholders; this software doesn’t host video.
