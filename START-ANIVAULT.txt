Install AniVault (Windows):
  https://github.com/PhantomBum/Anivault/releases
  Download AniVaultSetup.exe from the latest release, run it.

Developers (from repo root):
  cd desktop
  npm install
  npm start

Build installer:
  cd desktop
  npm run make
  (If EPERM: quit AniVault, then npm run make:clean)

Root index.html = AniList catalog in the browser only — use the desktop app to watch.
